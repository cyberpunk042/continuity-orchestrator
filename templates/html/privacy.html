1. Overview

Continuity Orchestrator is an open-source, self-hosted personal automation tool. It is designed for single-operator use ‚Äî there are no end users, no user accounts, and no public-facing data collection.

2. Data Collection

This project collects no personal data from visitors. Specifically:

  • No cookies are used
  • No analytics or tracking scripts are loaded
  • No third-party advertising or marketing services are integrated
  • No visitor data is stored, processed, or transmitted

The static pages served from this site are informational only. They display the current operational status of the system and are regenerated periodically.

3. Operator Data

The operator (the person who deployed and maintains this instance) stores their own configuration locally or in encrypted GitHub repository secrets. This includes:

  • API credentials for notification services (email, SMS, social media)
  • Operator contact information (email address, phone number)
  • Security passphrases for authentication

These secrets are never exposed on the public site. They are stored securely using environment variables, encrypted secrets, or local .env files.
